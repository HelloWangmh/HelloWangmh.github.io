shell学习笔记

发表于 2018-08-12

基础

  • 变量,等号和值之间不能出现空格
  • 引用一个变量的时候需要加上$
  • 命令替换 : 将命令输出赋给变量,通过反引号字符或$()

数学表达式

  • 格式 : $[operation] 例如 : $[1 + 1]
  • 浮点数计算 : bc命令 内建变量scale控制精度

退出脚本

  • 退出码 : $?变量保存着上一个命令退出后的状态码, 成功结束的状态码是0
  • exit命令可以为脚本指定一个退出状态码

If语句

  • test命令提供在if-then语句中测试不同条件的途径,如果test命令中条件成立,就会返回0状态码

while命令

控制循环

  • 终止循环 : break可以跳出当前循环, break n n指定了跳出循环的层级
  • 中止某次循环 : continue命令, continue n 同理

###命令行参数

  • 读取参数 : $0是程序名, $1是第一个参数,直到$9,以后都是${10}
  • basename会返回不包含路径的脚本吗 basename $0
  • $# 表示命令行参数个数
  • $* 会把所有参数当做一个单词保存
  • $@ 会把所有参数当做同一字符串的多个独立的单词
  • shift 命令能够移动命令行参数

read命令

控制脚本

  • Linux利用信号与运行在系统中进程进行通信
  • 可以在脚本中用trap命令捕获信号
  • 在命令后加上 & 可以以后台模式运行脚本,但是这样终端退出的话,后台进程也会退出
  • nohup命令可以通过解除终端和进程的关联,来让一直让脚本在后台运行
